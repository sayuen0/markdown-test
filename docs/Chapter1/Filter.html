フィルターとprefilter

フィルターとprefilter

まず始めにフィルターを定義する。本稿では集合上のフィルターを扱い、一般に真のフィルターと呼ばれるものを考える。

定義 X{ X }Xを集合とする。部分集合の族F⊂2X,≠∅{ \mathscr{F}\subset 2^{X}, \neq \emptyset }F⊂2X,̸​=∅は空でないとする。更にF{ \mathscr{F} }Fは以下を満たすとする。

  1. ∅∉F{ \emptyset\notin\mathscr{F} }∅∈/​Fである。
  2. V∈F,V⊂W{ V\in\mathscr{F}, V\subset W }V∈F,V⊂WならW∈F{ W\in\mathscr{F} }W∈Fが成り立つ。
  3. V,W∈F{ V, W\in\mathscr{F} }V,W∈FならV∩W∈F{ V\cap W\in\mathscr{F} }V∩W∈Fが成り立つ。

このときF{ \mathscr{F} }FをX{ X }Xのフィルター（filter）と呼ぶ。

例を挙げるとキリがないので重要なものを一つだけ挙げよう。

例 X{ X }Xを無限集合とする。X∖S{ X\setminus S }X∖Sが有限集合となるようなS⊂X{ S\subset X }S⊂X全体をフレシェ（Fréchet）フィルターと呼び、FC{ \mathcal{FC} }FCで表す。

この他にも位相空間X{ X }Xにおいてある集合A⊂X{ A\subset X }A⊂X（あるいは一点x∈X{ x\in X }x∈X）を内部に含む集合全体はフィルターを成す。ここからフィルターは近傍系の一般化であることが分かる。というよりもむしろ、位相はフィルターの場に遠近感を加えたものと捉えることができる。

次に、フィルターの作り方について学ぶ。色々と方法はあるが、以下の構成法が基本的だろう。

定義 部分集合の族B⊂2X,≠∅{ \mathscr{B}\subset 2^{X}, \neq \emptyset }B⊂2X,̸​=∅は空でないとする。更にB{ \mathscr{B} }Bは以下を満たすとする。

  1. ∅∉B{ \emptyset\notin\mathscr{B} }∅∈/​Bである。
  2. A,B∈B{ A, B\in\mathscr{B} }A,B∈Bなら、あるC∈B{ C\in\mathscr{B} }C∈Bが存在してC⊂A∩B{ C\subset A\cap B }C⊂A∩Bが成り立つ。

このときB{ \mathscr{B} }BをX{ X }Xのプレフィルター（prefilter）と呼ぶ。

注意 本稿ではprefilterと書くが、普通はフィルター基（filter base）と呼ぶ。prefilterという語を採用した理由は、filterとfilter baseでは個人的に視認性が悪いからである。prefilterと調べても出てこないので悪しからず。

  • filterはprefilterである。

命題 B⊂2X{ \mathscr{B}\subset 2^{X} }B⊂2Xはprefilterとする。このときB{ \mathscr{B} }Bを含む最小のフィルターが存在する。

（証明）各B∈B{ B\in\mathscr{B} }B∈BについてB{ B }Bより大きな集合を全て集めたもの

F={F:∃B∈B,B⊂F} \displaystyle \mathscr{F}=\lbrace F : \exists B\in\mathscr{B}, B\subset F \rbrace F={F:∃B∈B,B⊂F}

がB{ \mathscr{B} }Bを含む最小のフィルターとなる。実際∅∉B{ \emptyset\notin\mathscr{B} }∅∈/​Bより∅∉F{ \emptyset\notin\mathscr{F} }∅∈/​Fであり、V∈F,V⊂W{ V\in\mathscr{F}, V\subset W }V∈F,V⊂WについてB∈B{ B\in\mathscr{B} }B∈Bが存在してB⊂V⊂W{ B\subset V\subset W }B⊂V⊂Wが成り立つからW∈F{ W\in\mathscr{F} }W∈Fが従う。V,W∈F{ V, W\in\mathscr{F} }V,W∈FとするとA,B∈B{ A, B\in\mathscr{B} }A,B∈Bが存在してA⊂V,B⊂W{ A\subset V, B\subset W }A⊂V,B⊂Wとなるが、ここでprefilterの定義より、あるC∈B{ C\in\mathscr{B} }C∈Bが存在してC⊂A∩B⊂V∩W{ C\subset A\cap B\subset V\cap W }C⊂A∩B⊂V∩Wとなる。つまりV∩W∈F{ V\cap W\in\mathscr{F} }V∩W∈Fを得る。最小性は明らかだろう。□{ \square }□

最小性から一意性が従うので、フィルターの生成を定義することができる。

定義 prefilterB{ \mathscr{B} }Bを含む最小のフィルターを⟨B⟩{ \langle \mathscr{B} \rangle }⟨B⟩で表し、B{ \mathscr{B} }Bが生成するフィルターと呼ぶ。具体的には

⟨B⟩={F:∃B∈B,B⊂F} \displaystyle \langle \mathscr{B} \rangle = \lbrace F : \exists B\in\mathscr{B} , B\subset F \rbrace ⟨B⟩={F:∃B∈B,B⊂F}

で実現される。

  • filterはprefilterとして生成するフィルターと等しい。

定義 部分集合A⊂X,≠∅{ A\subset X, \neq \emptyset }A⊂X,̸​=∅は空でないとする。このとき⟨A⟩:=⟨{A}⟩{ \langle A \rangle:=\langle \lbrace A \rbrace \rangle }⟨A⟩:=⟨{A}⟩で表されるフィルターを単項フィルター（principal filter）と呼ぶ。特にx∈X{ x\in X }x∈XについてA={x}{ A=\lbrace x \rbrace }A={x}のとき⟨x⟩:=⟨{x}⟩=⟨{{x}}⟩{ \langle x \rangle:=\langle \lbrace x \rbrace \rangle=\langle \lbrace \lbrace x \rbrace \rbrace \rangle }⟨x⟩:=⟨{x}⟩=⟨{{x}}⟩を点フィルター（point filter）と呼ぶ。
