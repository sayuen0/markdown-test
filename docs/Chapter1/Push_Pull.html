push-forwardとpull-back

集合間の写像が与えられたとき、定義域のフィルターを値域へ押し出したり、あるいは値域のフィルターを定義域へ引き戻したりする操作を考えたくなる。このような操作はフィルターの生成を用いて定義されるが、自然な条件で自然な対応が付く。

命題 X,Y{ X, Y }X,Yを集合とし、A⊂2X{ \mathscr{A}\subset 2^{X} }A⊂2XをX{ X }Xのprefilterとする。写像f​:X→Y{ f\colon X\rightarrow Y }f:X→Yについて

f(A):={f(V)⊂Y:V∈A}⊂2Y \displaystyle f( \mathscr{A} ):=\lbrace f( V )\subset Y : V\in\mathscr{A} \rbrace\subset 2^{Y} f(A):={f(V)⊂Y:V∈A}⊂2Y

はY{ Y }Yのprefilterとなる。

（証明）V≠∅{ V\neq\emptyset }V̸​=∅よりf(V)≠∅{ f( V )\neq\emptyset }f(V)̸​=∅である。V,W∈A{ V, W\in\mathscr{A} }V,W∈Aについてf(V),f(W)∈f(A){ f( V ), f( W )\in f( \mathscr{A} ) }f(V),f(W)∈f(A)を考える。A{ \mathscr{A} }AはprefilterであるからA∈A{ A\in\mathscr{A} }A∈AをA⊂V∩W{ A\subset V\cap W }A⊂V∩Wとなるように取れる。f(A)∈f(A){ f( A )\in f( \mathscr{A} ) }f(A)∈f(A)かつf(A)⊂f(V∩W)⊂f(V)∩f(W){ f( A )\subset f( V\cap W )\subset f( V )\cap f( W ) }f(A)⊂f(V∩W)⊂f(V)∩f(W)より、f(A){ f( \mathscr{A} ) }f(A)はprefilterである。□{ \square }□

定義 特にA=F{ \mathscr{A}=\mathscr{F} }A=Fがフィルターであるとき、

f∗F:=⟨f(F)⟩ \displaystyle f_{\ast}\mathscr{F}:=\langle f( \mathscr{F} ) \rangle f∗​F:=⟨f(F)⟩

をF{ \mathscr{F} }Fのpush-forwardと呼ぶ。

命題 X,Y{ X, Y }X,Yを集合とし、B⊂2Y{ \mathscr{B}\subset 2^{Y} }B⊂2YをY{ Y }Yのprefilterとする。写像f​:X→Y{ f\colon X\rightarrow Y }f:X→Yについて、任意のB∈B{ B\in\mathscr{B} }B∈Bがf−1(B)≠∅{ f^{-1}( B )\neq\emptyset }f−1(B)̸​=∅を満たすとする。このとき

f−1(B):={f−1(W)⊂X:W∈B}⊂2X \displaystyle f^{-1}( \mathscr{B} ):=\lbrace f^{-1}( W )\subset X : W\in\mathscr{B} \rbrace\subset 2^{X} f−1(B):={f−1(W)⊂X:W∈B}⊂2X

はX{ X }Xのprefilterとなる。

（証明）定義よりf−1(B){ f^{-1}( \mathscr{B} ) }f−1(B)は空集合を含まない。V,W∈B{ V, W\in\mathscr{B} }V,W∈Bについてf−1(V),f−1(W)∈f−1(B){ f^{-1}( V ), f^{-1}( W )\in f^{-1}( \mathscr{B} ) }f−1(V),f−1(W)∈f−1(B)を考える。B{ \mathscr{B} }BはprefilterであるからB∈B{ B\in\mathscr{B} }B∈BをB⊂V∩W{ B\subset V\cap W }B⊂V∩Wとなるように取れる。f−1(B)⊂f−1(V∩W)=f−1(V)∩f−1(W){ f^{-1}( B )\subset f^{-1}( V\cap W )=f^{-1}( V )\cap f^{-1}( W ) }f−1(B)⊂f−1(V∩W)=f−1(V)∩f−1(W)かつf−1(B)∈f−1(B){ f^{-1}( B )\in f^{-1}( \mathscr{B} ) }f−1(B)∈f−1(B)より、f−1(B){ f^{-1}( \mathscr{B} ) }f−1(B)はprefilterである。□{ \square }□

定義 特にB=G{ \mathscr{B}=\mathscr{G} }B=Gがフィルターであるとき、

f∗G:=⟨f−1(G)⟩ \displaystyle f^{\ast}\mathscr{G}:=\langle f^{-1}( \mathscr{G} ) \rangle f∗G:=⟨f−1(G)⟩

をG{ \mathscr{G} }Gのpull-backと呼ぶ。

  • pull-backが定義できるG{ \mathscr{G} }Gには条件がある。

以下でこれらの性質を示す。まずフィルターの生成という観点から言うと、push-forwardは生成について可換である。

命題 f​:X→Y{ f\colon X\rightarrow Y }f:X→Yを写像とする。A⊂2X{ \mathscr{A}\subset 2^{X} }A⊂2XをX{ X }Xのprefilterとすると、

f∗⟨A⟩=⟨f(A)⟩ \displaystyle f_{\ast}\langle \mathscr{A} \rangle=\langle f( \mathscr{A} ) \rangle f∗​⟨A⟩=⟨f(A)⟩

が成り立つ。

（証明）f∗⟨A⟩⊃f(⟨A⟩)⊃f(A){ f_{\ast}\langle \mathscr{A} \rangle\supset f( \langle \mathscr{A} \rangle )\supset f( \mathscr{A} ) }f∗​⟨A⟩⊃f(⟨A⟩)⊃f(A)なので、生成の最小性よりf∗⟨A⟩⊃⟨f(A)⟩{ f_{\ast}\langle \mathscr{A} \rangle\supset\langle f( \mathscr{A} ) \rangle }f∗​⟨A⟩⊃⟨f(A)⟩が従う。よって逆を示せば良い。左辺から元を取りF∈f∗⟨A⟩{ F\in f_{\ast}\langle \mathscr{A} \rangle }F∈f∗​⟨A⟩とする。あるG∈⟨A⟩{ G\in\langle \mathscr{A} \rangle }G∈⟨A⟩が存在してf(G)⊂F{ f( G )\subset F }f(G)⊂Fが成り立つ。更にG{ G }Gについて、あるH∈A{ H\in\mathscr{A} }H∈Aが存在してH⊂G{ H\subset G }H⊂Gが成り立つ。よってf(H)⊂f(G)⊂F{ f( H )\subset f( G )\subset F }f(H)⊂f(G)⊂FだからF∈⟨f(A)⟩{ F\in\langle f( \mathscr{A} ) \rangle }F∈⟨f(A)⟩を得る。□{ \square }□

push-forwardの合成はpush-forwardである。

命題 X,Y,Z{ X, Y, Z }X,Y,Zを集合とし、f​:X→Y,g​:Y→Z{ f\colon X\rightarrow Y, g\colon Y\rightarrow Z }f:X→Y,g:Y→Zを写像とすると、

(g∘f)∗=g∗∘f∗ \displaystyle ( g\circ f )_{\ast}=g_{\ast}\circ f_{\ast} (g∘f)∗​=g∗​∘f∗​

が成り立つ。

（証明）F⊂2X{ \mathscr{F}\subset 2^{X} }F⊂2XをX{ X }Xのフィルターとする。(g∘f)∗F=g∗(f∗F){ ( g\circ f )_{\ast}\mathscr{F}=g_{\ast}( f_{\ast}\mathscr{F} ) }(g∘f)∗​F=g∗​(f∗​F)を示せば良い。左辺から元を取りF∈(g∘f)∗F{ F\in( g\circ f )_{\ast}\mathscr{F} }F∈(g∘f)∗​Fとする。あるG∈F{ G\in\mathscr{F} }G∈Fが存在してg∘f(G)⊂F{ g\circ f( G )\subset F }g∘f(G)⊂Fが成り立つ。g∘f(G)=g(f(G)){ g\circ f( G )=g( f( G ) ) }g∘f(G)=g(f(G))だからf(G)∈f(F)⊂f∗F{ f( G )\in f( \mathscr{F} )\subset f_{\ast}\mathscr{F} }f(G)∈f(F)⊂f∗​FよりF∈g∗(f∗(F)){ F\in g_{\ast}( f_{\ast}( \mathscr{F} ) ) }F∈g∗​(f∗​(F))を得る。逆にF∈g∗(f∗F){ F\in g_{\ast}( f_{\ast}\mathscr{F} ) }F∈g∗​(f∗​F)とすると、あるG∈f∗F{ G\in f_{\ast}\mathscr{F} }G∈f∗​Fが存在してg(G)⊂F{ g( G )\subset F }g(G)⊂Fが成り立つ。同様にあるH∈F{ H\in\mathscr{F} }H∈Fが存在してf(H)⊂G{ f( H )\subset G }f(H)⊂Gが成り立つから、(g∘f)(H)⊂g(G)⊂F{ ( g\circ f )( H )\subset g( G )\subset F }(g∘f)(H)⊂g(G)⊂Fを得る。故にF∈(g∘f)∗F{ F\in ( g\circ f )_{\ast}\mathscr{F} }F∈(g∘f)∗​Fとなる。□{ \square }□

push-forwardとpull-backの間には以下の関係がある。

命題 f​:X→Y{ f\colon X\rightarrow Y }f:X→Yを写像とする。F⊂2X,G⊂2Y{ \mathscr{F}\subset 2^{X}, \mathscr{G}\subset 2^{Y} }F⊂2X,G⊂2YをX,Y{ X, Y }X,Yのフィルターとする。以下が成り立つ。

  1. f∗F{ f_{\ast}\mathscr{F} }f∗​Fのpull-backは常に定義できてF⊂f∗(f∗F){ \mathscr{F}\subset f^{\ast}( f_{\ast}\mathscr{F} ) }F⊂f∗(f∗​F)を満たす。
  2. 任意のG∈G{ G\in\mathscr{G} }G∈Gがf−1(G)≠∅{ f^{-1}( G )\neq\emptyset }f−1(G)̸​=∅を満たすなら（G{ \mathscr{G} }Gのpull-backが定義できるなら）G⊂f∗(f∗G){ \mathscr{G}\subset f_{\ast}( f^{\ast}\mathscr{G} ) }G⊂f∗​(f∗G)が成り立つ。

（証明）上は自明。下はG∈G{ G\in\mathscr{G} }G∈Gに対し、f(f−1(G))⊂G{ f( f^{-1}( G ) )\subset G }f(f−1(G))⊂GよりG∈f∗(f∗G){ G\in f_{\ast}( f^{\ast}\mathscr{G} ) }G∈f∗​(f∗G)を得る。□{ \square }□

写像が全射のときpush-forwardはフィルターの像と一致する。

補題 f​:X→Y{ f\colon X\rightarrow Y }f:X→Yは全射とする。F⊂2X{ \mathscr{F}\subset 2^{X} }F⊂2XをX{ X }Xのフィルターとすると、f∗F=f(F){ f_{\ast}\mathscr{F}=f( \mathscr{F} ) }f∗​F=f(F)が成り立つ。特にf(F){ f( \mathscr{F} ) }f(F)はフィルターである。

（証明）左辺から元を取りG∈f∗F{ G\in f_{\ast}\mathscr{F} }G∈f∗​Fとする。あるF∈F{ F\in\mathscr{F} }F∈Fが存在してf(F)⊂G{ f( F )\subset G }f(F)⊂Gが成り立つ。F⊂f−1(f(F))⊂f−1(G){ F\subset f^{-1}( f( F ) )\subset f^{-1}( G ) }F⊂f−1(f(F))⊂f−1(G)よりf−1(G)∈F{ f^{-1}( G )\in\mathscr{F} }f−1(G)∈Fだが、f{ f }fは全射なのでG=f(f−1(G))∈f(F){ G=f( f^{-1}( G ) )\in f( \mathscr{F} ) }G=f(f−1(G))∈f(F)を得る。（一般にはf(f−1(G))⊂G{ f( f^{-1}( G ) )\subset G }f(f−1(G))⊂Gなので成立しない。）□{ \square }□

写像が全射のときはpull-backも定義できる。更にpull-backがフィルターの逆像と一致するには、写像が全単射であることが十分である。

補題 f​:X→Y{ f\colon X\rightarrow Y }f:X→Yは全単射とする。G⊂2Y{ \mathscr{G}\subset 2^{Y} }G⊂2YをY{ Y }Yのフィルターとすると、f∗G=f−1(G){ f^{\ast}\mathscr{G}=f^{-1}( \mathscr{G} ) }f∗G=f−1(G)が成り立つ。特にf−1(G){ f^{-1}( \mathscr{G} ) }f−1(G)はフィルターである。

（証明）左辺から元を取りF∈f∗G{ F\in f^{\ast}\mathscr{G} }F∈f∗Gとする。f{ f }fは単射なのでF=f−1(f(F)){ F=f^{-1}( f( F ) ) }F=f−1(f(F))が成り立つから、f(F)∈G{ f( F )\in\mathscr{G} }f(F)∈Gを示せば良い。（一般にはF⊂f−1(f(F)){ F\subset f^{-1}( f( F ) ) }F⊂f−1(f(F))である。）F∈f∗G{ F\in f^{\ast}\mathscr{G} }F∈f∗Gより、あるG∈G{ G\in\mathscr{G} }G∈Gが存在してf−1(G)⊂F{ f^{-1}( G )\subset F }f−1(G)⊂Fが成り立つ。f(f−1(G))⊂f(F){ f( f^{-1}( G ) )\subset f( F ) }f(f−1(G))⊂f(F)となるが、f{ f }fは全射なのでG=f(f−1(G))⊂f(F){ G=f( f^{-1}( G ) )\subset f( F ) }G=f(f−1(G))⊂f(F)を得る。G∈G{ G\in\mathscr{G} }G∈Gからf(F)∈G{ f( F )\in\mathscr{G} }f(F)∈Gが従う。□{ \square }□

系 f​:X→Y{ f\colon X\rightarrow Y }f:X→Yは全単射とする。Φ(X),Φ(Y){ \Phi( X ), \Phi( Y ) }Φ(X),Φ(Y)はX,Y{ X, Y }X,Yのフィルター全体の集合とする。このとき

f∗∘f∗=idΦ(X),f∗∘f∗=idΦ(Y) \displaystyle \begin{alignedat}{2} f^{\ast}\circ f_{\ast}&=\mathrm{id}_{\Phi( X )}, &\qquad f_{\ast}\circ f^{\ast}&=\mathrm{id}_{\Phi( Y )} \end{alignedat} f∗∘f∗​​=idΦ(X)​,​f∗​∘f∗​=idΦ(Y)​​

が成り立つ。
