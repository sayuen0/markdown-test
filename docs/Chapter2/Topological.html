トポロジカルな収束構造

位相空間(X,O){ ( X, \mathcal{O} ) }(X,O)が自然な収束構造を持つことを述べよう。N⊂X{ N\subset X }N⊂Xがx∈X{ x\in X }x∈Xの近傍であるとは、あるU∈O{ U\in\mathcal{O} }U∈Oが存在してx∈U⊂N{ x\in U\subset N }x∈U⊂Nを満たすことだった。x{ x }xの近傍全体を近傍系と呼びN(x){ \mathfrak{N}( x ) }N(x)で表せば次が成り立つ。

  1. 任意のx∈X{ x\in X }x∈XについてX∈N(x){ X\in\mathfrak{N}( x ) }X∈N(x)であり、N∈N(x){ N\in\mathfrak{N}( x ) }N∈N(x)ならx∈N{ x\in N }x∈Nを満たす。
  2. N,M∈N(x){ N, M\in\mathfrak{N}( x ) }N,M∈N(x)ならN∩M∈N(x){ N\cap M\in\mathfrak{N}( x ) }N∩M∈N(x)である。
  3. N∈N(x){ N\in\mathfrak{N}( x ) }N∈N(x)かつN⊂M⊂X{ N\subset M\subset X }N⊂M⊂XならM∈N(x){ M\in\mathfrak{N}( x ) }M∈N(x)である。
  4. 任意のN∈N(x){ N\in\mathfrak{N}( x ) }N∈N(x)に対して、あるM∈N(x){ M\in\mathfrak{N}( x ) }M∈N(x)が存在して、y∈M{ y\in M }y∈MならN∈N(y){ N\in\mathfrak{N}( y ) }N∈N(y)が成り立つ。

逆にこれらを満たすN(x){ \mathfrak{N}( x ) }N(x)から位相を定義することもでき（Hausdorffの公理系）、その位相における近傍系と両立することが知られている。

さて1番目の前半と、2、3番目はほとんどフィルターの定義である。実際これをフィルターの定義とする流儀もあるが、本稿の定義においても近傍系はフィルターである。また1番目の後半は、収束空間の⟨x⟩→x{ \langle x \rangle\rightarrow x }⟨x⟩→xに対応していると考えられる。そこでF→x{ \mathscr{F}\rightarrow x }F→xをN(x)⊂F{ \mathfrak{N}( x )\subset\mathscr{F} }N(x)⊂Fで定めよう。この収束構造をtopological convergence structureと呼ぶ。

定義 収束空間X{ X }Xが位相的（topological）とは、収束構造がtopological convergence structureと一致することをいう。つまりあるX{ X }Xの位相O{ \mathcal{O} }Oが存在して、F→x{ \mathscr{F}\rightarrow x }F→xとN(x)⊂F{ \mathfrak{N}( x )\subset\mathscr{F} }N(x)⊂Fが同値となるときに、トポロジカルであるという。

注意 トポロジカルな収束構造とtopological convergence structureは同値の概念だが、出発点が違うという意味で区別している。

命題 X{ X }Xはトポロジカルな収束空間とする。Φ(X){ \Phi( X ) }Φ(X)をX{ X }Xのフィルター全体とする。このとき

N(x)=∩{F∈Φ(X):F→x} \displaystyle \mathfrak{N}( x )=\cap\lbrace \mathscr{F}\in\Phi( X ) : \mathscr{F}\rightarrow x \rbrace N(x)=∩{F∈Φ(X):F→x}

が成り立つ。

（証明）F→x{ \mathscr{F}\rightarrow x }F→xならN(x)⊂F{ \mathfrak{N}( x )\subset\mathscr{F} }N(x)⊂Fより左辺は右辺に含まれる。逆は近傍系がフィルターであることからN(x)→x{ \mathfrak{N}( x )\rightarrow x }N(x)→xより従う。□{ \square }□

  • 位相は近傍系より一意的に定まるから、収束空間がトポロジカルなときその位相は一意的である。

この左辺は収束空間がトポロジカルでなければ定義できないが、右辺は一般の収束空間で定義できることに気付くだろう。

定義 X{ X }Xを収束空間とする。X{ X }Xのフィルター全体をΦ(X){ \Phi( X ) }Φ(X)とする。x∈X{ x\in X }x∈Xに対し、x{ x }xに収束するフィルターの共通部分

Nx:=∩{F∈Φ(X):F→x} \displaystyle \mathscr{N}_{x}:=\cap\lbrace \mathscr{F}\in\Phi( X ) : \mathscr{F}\rightarrow x \rbrace Nx​:=∩{F∈Φ(X):F→x}

をx{ x }xの近傍フィルター（neighborhood filter）と呼び、この元をx{ x }xの近傍と呼ぶ。

  • トポロジカルな収束空間において、近傍フィルターは近傍系と一致し、各点の近傍は両者の意味で同一となる。

近傍系から位相を作る方法を思い出せば、位相的概念を収束構造に持ち込むことが出来る。例えば開集合は任意のx∈U{ x\in U }x∈UについてU∈N(x){ U\in\mathfrak{N}( x ) }U∈N(x)が成り立つ集合U⊂X{ U\subset X }U⊂Xである。そこで次を定める。

定義 X{ X }Xを収束空間とする。U⊂X{ U\subset X }U⊂Xとする。任意のx∈U{ x\in U }x∈UについてU∈Nx{ U\in\mathscr{N}_{x} }U∈Nx​が成り立つとき、U{ U }Uは開（open）であるという。

  • トポロジカルな収束空間において、収束構造としてopenであることは、位相構造として開集合であることと同値である。

補題 X,Y{ X, Y }X,Yをトポロジカルな収束空間とする。f​:X→Y{ f\colon X\rightarrow Y }f:X→Yを写像とする。以下は同値である。

  • f{ f }fは収束構造として連続である。
  • U⊂Y{ U\subset Y }U⊂Yがopenなら、f−1(U)⊂X{ f^{-1}( U )\subset X }f−1(U)⊂Xもopenである。

（証明）上から下は常に成り立つ。実際U⊂Y{ U\subset Y }U⊂Yをopenとすると、x∈f−1(U){ x\in f^{-1}( U ) }x∈f−1(U)についてf(x)∈U{ f( x )\in U }f(x)∈Uより、U∈Nf(x){ U\in\mathscr{N}_{f( x )} }U∈Nf(x)​が成り立つ。f−1(U)∈Nx{ f^{-1}( U )\in\mathscr{N}_{x} }f−1(U)∈Nx​を示したい。F→x{ \mathscr{F}\rightarrow x }F→xを取る。f{ f }fは連続だからf∗F→f(x){ f_{\ast}\mathscr{F}\rightarrow f( x ) }f∗​F→f(x)である。U{ U }UはopenだからU∈f∗F{ U\in f_{\ast}\mathscr{F} }U∈f∗​Fとなる。つまり、あるF∈F{ F\in\mathscr{F} }F∈Fが存在してf(F)⊂U{ f( F )\subset U }f(F)⊂Uとなる。F⊂f−1(U){ F\subset f^{-1}( U ) }F⊂f−1(U)よりf−1(U)∈F{ f^{-1}( U )\in\mathscr{F} }f−1(U)∈Fを得る。従ってf−1(U)∈Nx{ f^{-1}( U )\in\mathscr{N}_{x} }f−1(U)∈Nx​つまりf−1(U){ f^{-1}( U ) }f−1(U)はopenである。

下から上を示すのにトポロジカルであることが必要となる。X,Y{ X, Y }X,Yの位相をOX,OY{ \mathcal{O}_{X}, \mathcal{O}_{Y} }OX​,OY​とする。F→x{ \mathscr{F}\rightarrow x }F→xとする。f{ f }fが収束構造として連続であることを示すにはf∗F→f(x){ f_{\ast}\mathscr{F}\rightarrow f( x ) }f∗​F→f(x)つまりN(f(x))⊂f∗F{ \mathfrak{N}( f( x ) )\subset f_{\ast}\mathscr{F} }N(f(x))⊂f∗​Fを示せば良い。N∈N(f(x)){ N\in\mathfrak{N}( f( x ) ) }N∈N(f(x))を取ると、ある開集合V∈OY{ V\in\mathcal{O}_{Y} }V∈OY​が存在してf(x)∈V⊂N{ f( x )\in V\subset N }f(x)∈V⊂Nとなる。x∈f−1(V)⊂f−1(N){ x\in f^{-1}( V )\subset f^{-1}( N ) }x∈f−1(V)⊂f−1(N)だが、仮定よりf−1(V){ f^{-1}( V ) }f−1(V)は開集合、特にf−1(V)∈N(x){ f^{-1}( V )\in\mathfrak{N}( x ) }f−1(V)∈N(x)となる。故にf−1(N)∈N(x){ f^{-1}( N )\in\mathfrak{N}( x ) }f−1(N)∈N(x)だからf(f−1(N))⊂N{ f( f^{-1}( N ) )\subset N }f(f−1(N))⊂NよりN∈f∗N(x)⊂f∗F{ N\in f_{\ast}\mathfrak{N}( x )\subset f_{\ast}\mathscr{F} }N∈f∗​N(x)⊂f∗​Fが従う。□{ \square }□

定理 X{ X }Xを収束空間、Y{ Y }Yを位相空間とする。Y{ Y }Yはtopological convergence structureで収束空間とみなす。f​:X→Y{ f\colon X\rightarrow Y }f:X→Yが収束空間としての同型を与えているなら、X{ X }Xはトポロジカルであり、f{ f }fは同相写像となる。

（証明）OY{ \mathcal{O}_{Y} }OY​をY{ Y }Yの位相とする。必然的に

OX:={U⊂X:f(U)∈OY} \displaystyle \mathcal{O}_{X}:=\lbrace U\subset X : f( U )\in\mathcal{O}_{Y} \rbrace OX​:={U⊂X:f(U)∈OY​}

と置くしかない。f{ f }fは全単射なので、これがX{ X }X上の位相を定めることは明らかだろう。F→x{ \mathscr{F}\rightarrow x }F→xとN(x)⊂F{ \mathfrak{N}( x )\subset\mathscr{F} }N(x)⊂Fが同値となることを示せば良い。

F→x{ \mathscr{F}\rightarrow x }F→xとする。N∈N(x){ N\in\mathfrak{N}( x ) }N∈N(x)とすると、あるU∈OX{ U\in\mathcal{O}_{X} }U∈OX​が存在してx∈U⊂N{ x\in U\subset N }x∈U⊂Nが成り立つ。f(x)∈f(U)⊂f(N){ f( x )\in f( U )\subset f( N ) }f(x)∈f(U)⊂f(N)だが、f(U){ f( U ) }f(U)は開集合なのでf(N)∈N(f(x)){ f( N )\in\mathfrak{N}( f( x ) ) }f(N)∈N(f(x))である。ここでf{ f }fは連続なのでf(F)→f(x){ f( \mathscr{F} )\rightarrow f( x ) }f(F)→f(x)が成り立つ。よってN(f(x))⊂f(F){ \mathfrak{N}( f( x ) )\subset f( \mathscr{F} ) }N(f(x))⊂f(F)が成り立つ。つまりf(N)∈f(F){ f( N )\in f( \mathscr{F} ) }f(N)∈f(F)を得る。f{ f }fは全単射だからN∈F{ N\in\mathscr{F} }N∈Fである。

N(x)⊂F{ \mathfrak{N}( x )\subset\mathscr{F} }N(x)⊂Fとする。F→x{ \mathscr{F}\rightarrow x }F→xを示したい。f{ f }fは同型なのでf(F)→f(x){ f( \mathscr{F} )\rightarrow f( x ) }f(F)→f(x)を示せば良い。更にY{ Y }YはトポロジカルだからN(f(x))⊂f(F){ \mathfrak{N}( f( x ) )\subset f( \mathscr{F} ) }N(f(x))⊂f(F)を示せばよい。N∈N(f(x)){ N\in\mathfrak{N}( f( x ) ) }N∈N(f(x))とする。ある開集合V∈OY{ V\in\mathcal{O}_{Y} }V∈OY​が存在してf(x)∈V⊂N{ f( x )\in V\subset N }f(x)∈V⊂Nとなる。従ってx∈f−1(V)⊂f−1(N){ x\in f^{-1}( V )\subset f^{-1}( N ) }x∈f−1(V)⊂f−1(N)となる。定義よりf−1(V)∈OX{ f^{-1}( V )\in\mathcal{O}_{X} }f−1(V)∈OX​だから、仮定よりf−1(N)∈N(x)⊂F{ f^{-1}( N )\in\mathfrak{N}( x )\subset\mathscr{F} }f−1(N)∈N(x)⊂Fを得る。故にN=f(f−1(N))∈f(F){ N=f( f^{-1}( N ) )\in f( \mathscr{F} ) }N=f(f−1(N))∈f(F)が分かる。

f{ f }fが同相写像となることは、前の補題より明らかだろう。□{ \square }□
