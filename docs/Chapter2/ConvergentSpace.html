収束空間

収束空間

収束空間の研究はCartanから約10年後にChoquetなどにより始められた。収束空間の定義は様々だが、Eva Lowen-ColebundersのFunction Classes of Cauchy Continuous Maps（1989）で言及されている（らしい）定義を採用する。

定義 X{ X }Xを集合とする。各点x∈X{ x\in X }x∈XについてX{ X }Xのフィルターの族λ(x){ \lambda( x ) }λ(x)が与えられているとする。λ​:x↦λ(x){ \lambda\colon x\mapsto\lambda( x ) }λ:x↦λ(x)が以下を満たすとき、(X,λ){ ( X, \lambda) }(X,λ)を収束空間（convergent space）と呼ぶ。

  1. ⟨x⟩∈λ(x){ \langle x \rangle\in\lambda( x ) }⟨x⟩∈λ(x)である。
  2. F∈λ(x){ \mathscr{F}\in\lambda( x ) }F∈λ(x)とする。フィルターG⊂2X{ \mathscr{G}\subset 2^{X} }G⊂2XについてF⊂G{ \mathscr{F}\subset\mathscr{G} }F⊂GならG∈λ(x){ \mathscr{G}\in\lambda( x ) }G∈λ(x)を満たす。
  3. F,G∈λ(x){ \mathscr{F}, \mathscr{G}\in\lambda( x ) }F,G∈λ(x)ならF∩G∈λ(x){ \mathscr{F}\cap\mathscr{G}\in\lambda( x ) }F∩G∈λ(x)である。

フィルターF⊂2X{ \mathscr{F}\subset 2^{X} }F⊂2XについてF∈λ(x){ \mathscr{F}\in\lambda( x ) }F∈λ(x)であることをF→x{ \mathscr{F}\rightarrow x }F→xと書けばλ{ \lambda }λを明示せずに収束空間を定義できる。この意味で上記を満たす関係→{ \rightarrow }→を収束構造（convergent structure）とも呼ぶ。（X{ X }Xのフィルター全体をΦ(X){ \Phi( X ) }Φ(X)とすれば、Φ(X)×X{ \Phi( X )\times X }Φ(X)×Xの部分集合としての「関係」である。）またF→x{ \mathscr{F}\rightarrow x }F→xのことを、F{ \mathscr{F} }Fがx{ x }xに収束するともいう。x{ x }xはF{ \mathscr{F} }Fの極限点や収束先ともいう。

位相空間の近傍系x↦N(x){ x\mapsto\mathfrak{N}( x ) }x↦N(x)は収束構造を定める。これについては後で詳しく扱うとして、ここではもう少し素朴な例を挙げよう。

例 有向グラフ（directed graph）とは、頂点の集合V{ V }Vと、向き付けられた辺の集合E⊂V×V{ E\subset V\times V }E⊂V×Vの組(V,E){ ( V, E ) }(V,E)である。（有向辺(v,u)∈E{ (v, u )\in E }(v,u)∈Eは始点v{ v }vから終点u{ u }uへの矢印で表す。）有効グラフが反射的（reflexible）とは、任意の頂点v∈V{ v\in V }v∈Vについて(v,v)∈E{ ( v, v )\in E }(v,v)∈Eが成り立つことを言う。

さて反射的有向グラフ(V,E){ ( V, E ) }(V,E)において、頂点v∈V{ v\in V }v∈Vについてv{ v }vから出る矢印の終点全体

v→:={u∈V:(v,u)∈E} \displaystyle \overrightarrow{v}:=\lbrace u\in V : ( v, u )\in E \rbrace v:={u∈V:(v,u)∈E}

を考える。このとき収束構造F→v{ \mathscr{F}\rightarrow v }F→vをv→∈F{ \overrightarrow{v}\in\mathscr{F} }v∈Fで定めることができる。実際、(v,v)∈E{ ( v, v )\in E }(v,v)∈Eよりv∈v→{ v\in\overrightarrow{v} }v∈vだからv→∈⟨v⟩{ \overrightarrow{v}\in\langle v \rangle }v∈⟨v⟩である。F→v,F⊂G{ \mathscr{F}\rightarrow v, \mathscr{F}\subset\mathscr{G} }F→v,F⊂Gとすれば、v→∈F⊂G{ \overrightarrow{v}\in\mathscr{F}\subset\mathscr{G} }v∈F⊂GよりG→v{ \mathscr{G}\rightarrow v }G→vである。v→∈F,G{ \overrightarrow{v}\in\mathscr{F}, \mathscr{G} }v∈F,Gならv→∈F∩G{ \overrightarrow{v}\in\mathscr{F}\cap\mathscr{G} }v∈F∩Gも成り立つ。このようにして反射的有向グラフは収束空間となる。この収束構造において、単項フィルター⟨v→⟩{ \langle \overrightarrow{v} \rangle }⟨v⟩はv{ v }vに収束する。

念頭にあるのは収束空間のカテゴリーである。次は射について考えてみよう。

定義 X,Y{ X, Y }X,Yを収束空間、f​:X→Y{ f\colon X\rightarrow Y }f:X→Yを写像とする。x∈X{ x\in X }x∈Xとする。任意のフィルターF⊂2X{ \mathscr{F}\subset 2^{X} }F⊂2Xについて、F→x{ \mathscr{F}\rightarrow x }F→xならf∗F→f(x){ f_{\ast}\mathscr{F}\rightarrow f( x ) }f∗​F→f(x)が成り立つとき、写像f{ f }fはx{ x }xで連続（continuous）であるという。任意の点x∈X{ x\in X }x∈Xでf{ f }fが連続なとき、f{ f }fは連続であるという。

命題 X,Y,Z{ X, Y, Z }X,Y,Zを収束空間、f​:X→Y,g​:Y→Z{ f\colon X\rightarrow Y, g\colon Y\rightarrow Z }f:X→Y,g:Y→Zを写像とする。f,g{ f, g }f,gが連続ならg∘f{ g\circ f }g∘fも連続である。

（証明）f,g{ f, g }f,gは連続とする。F→x{ \mathscr{F}\rightarrow x }F→xに対しf∗F→f(x){ f_{\ast}\mathscr{F}\rightarrow f( x ) }f∗​F→f(x)であるからg∗(f∗F)→g(f(x)){ g_{\ast}( f_{\ast}\mathscr{F} )\rightarrow g( f( x ) ) }g∗​(f∗​F)→g(f(x))を得る。g∗∘f∗=(g∘f)∗{ g_{\ast}\circ f_{\ast}=( g\circ f )_{\ast} }g∗​∘f∗​=(g∘f)∗​より、g∘f{ g\circ f }g∘fも連続となる。□{ \square }□

例 反射的有向グラフの例で連続写像を見てみよう。f​:(V1,E1)→(V2,E2){ f\colon( V_{1}, E_{1} )\rightarrow( V_{2}, E_{2} ) }f:(V1​,E1​)→(V2​,E2​)をグラフ間の写像として、v∈V1{ v\in V_{1} }v∈V1​とする。f{ f }fがv{ v }vで連続であるとしよう。このとき単項フィルター⟨v→⟩{ \langle \overrightarrow{v} \rangle }⟨v⟩はv{ v }vに収束するから⟨v→⟩→v{ \langle \overrightarrow{v } \rangle\rightarrow v }⟨v⟩→vである。故にf∗⟨v→⟩→f(v){ f_{\ast}\langle \overrightarrow{v} \rangle\rightarrow f( v ) }f∗​⟨v⟩→f(v)が成り立つ。つまりf(v)→∈f∗⟨v→⟩{ \overrightarrow{f( v )}\in f_{\ast}\langle \overrightarrow{v} \rangle }f(v)​∈f∗​⟨v⟩であって、従ってf(v→)⊂f(v)→{ f( \overrightarrow{v} )\subset \overrightarrow{f( v )} }f(v)⊂f(v)​が成り立つ。逆にf(v→)⊂f(v)→{ f( \overrightarrow{v} )\subset\overrightarrow{f( v )} }f(v)⊂f(v)​が成り立つなら、F→v{ \mathscr{F}\rightarrow v }F→vについてv→∈F{ \overrightarrow{v}\in\mathscr{F} }v∈Fよりf(v→)∈f(F)⊂f∗F{ f( \overrightarrow{v} )\in f( \mathscr{F} )\subset f_{\ast}\mathscr{F} }f(v)∈f(F)⊂f∗​Fである。よってf(v)→∈f∗F{ \overrightarrow{f( v )}\in f_{\ast}\mathscr{F} }f(v)​∈f∗​Fとなりf{ f }fはv{ v }vで連続となる。従って次が成り立つ。

定理 f​:(V1,E1)→(V2,E2){ f\colon( V_{1}, E_{1} )\rightarrow( V_{2}, E_{2} ) }f:(V1​,E1​)→(V2​,E2​)を反射的有向グラフの間の写像とする。v,u∈V1{ v, u\in V_{1} }v,u∈V1​とする。以下は同値である。

  • f{ f }fはv{ v }vで連続である。
  • f(v→)⊂f(v)→{ f( \overrightarrow{v} )\subset\overrightarrow{f( v )} }f(v)⊂f(v)​が成り立つ。

更に以下も同値である。

  • f{ f }fは連続である。
  • (v,u)∈E1{ ( v, u )\in E_{1} }(v,u)∈E1​なら(f(v),f(u))∈E2{ ( f( v ), f( u ) )\in E_{2} }(f(v),f(u))∈E2​が成り立つ。（つまりf{ f }fはグラフ準同型である。）

カテゴリー論的に言えば、反射的有向グラフとグラフ準同型の為す圏は、収束空間と連続写像の為す圏に忠実充満に埋め込める。この意味で連続性は自然な定義と言える。実はそれだけでなく、位相空間の圏もまた忠実充満に埋め込める。後で述べるが、収束空間の圏はカルテシアン閉であり、特にX{ X }XからY{ Y }Yへの連続写像全体C(X,Y){ C( X, Y ) }C(X,Y)もまた「自然な」収束構造を持つ。位相空間の圏はこの性質を持たない（反射的有向グラフの圏も持たない？）ため、我々が収束空間について考える良い動機付けになっている。(なお位相空間が特別な場合には、コンパクト開位相という自然な位相が入る。これはこれで考察する意義がある。)

定義 収束空間の間の写像f​:X→Y{ f\colon X\rightarrow Y }f:X→Yが全単射でf{ f }f及びf−1{ f^{-1} }f−1が連続のとき、X{ X }XとY{ Y }Yは収束空間として同型（isomorphic）であるという。
