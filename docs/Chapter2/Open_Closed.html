位相的概念とその性質

位相的概念とその性質

今回は閉集合について考えてみよう。せっかく収束概念があるので以下のように考える。

定義 X{ X }Xを収束空間とする。Φ(X){ \Phi( X ) }Φ(X)をX{ X }Xのフィルター全体とする。A⊂X{ A\subset X }A⊂Xに対し、A{ A }Aの閉包（closure）を

cl(A):={x∈X:∃F∈Φ(X),F→x,A∈F} \displaystyle \mathrm{cl}( A ):=\lbrace x\in X : \exists\mathscr{F}\in\Phi( X ), \mathscr{F}\rightarrow x, A\in\mathscr{F} \rbrace cl(A):={x∈X:∃F∈Φ(X),F→x,A∈F}

で定める。つまりA{ A }Aを含むフィルターの極限点全体をcl(A){ \mathrm{cl}( A ) }cl(A)とする。

この定義が自然なものであると確認するには、後で述べるフィルターの制限や収束空間の部分空間についての議論が必要となる。今はA{ A }Aの収束列の収束先という類推で納得しておく。

上のcl{ \mathrm{cl} }clはKatětov閉包と呼ばれるらしい。

命題 閉包は以下を満たす。

  1. cl(∅)=∅{ \mathrm{cl}( \emptyset )=\emptyset }cl(∅)=∅である。
  2. A⊂cl(A){ A\subset\mathrm{cl}( A ) }A⊂cl(A)である。
  3. cl(A∪B)=cl(A)∪cl(B){ \mathrm{cl}( A\cup B )=\mathrm{cl}( A )\cup\mathrm{cl}( B ) }cl(A∪B)=cl(A)∪cl(B)が成り立つ。

3番目から特にA⊂B{ A\subset B }A⊂Bならcl(A)⊂cl(B){ \mathrm{cl}( A )\subset\mathrm{cl}( B ) }cl(A)⊂cl(B)が成り立つ。

（証明）フィルターは空集合を含まないからcl(∅)=∅{ \mathrm{cl}( \emptyset )=\emptyset }cl(∅)=∅である。

x∈A{ x\in A }x∈Aとすると⟨x⟩→x{ \langle x \rangle\rightarrow x }⟨x⟩→xかつA∈⟨x⟩{ A\in\langle x \rangle }A∈⟨x⟩が成り立つ。よってa∈cl(A){ a\in\mathrm{cl}( A ) }a∈cl(A)が従う。

x∈cl(A∪B){ x\in\mathrm{cl}( A\cup B ) }x∈cl(A∪B)とする。あるフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してA∪B∈F{ A\cup B\in\mathscr{F} }A∪B∈Fとなる。A∉F{ A\notin\mathscr{F} }A∈/​FのときF∖A={V:∃F∈F,F∖A⊂V}{ \mathscr{F}_{\setminus A}=\lbrace V : \exists F\in\mathscr{F}, F\setminus A\subset V \rbrace }F∖A​={V:∃F∈F,F∖A⊂V}はF{ \mathscr{F} }Fを含むフィルターである。よってF∖A→x{ \mathscr{F}_{\setminus A}\rightarrow x }F∖A​→xであり、一方A∪B∖A⊂B{ A\cup B\setminus A\subset B }A∪B∖A⊂BよりB∈F∖A{ B\in\mathscr{F}_{\setminus A} }B∈F∖A​である。従ってx∈cl(B)⊂cl(A)∪cl(B){ x\in\mathrm{cl}( B )\subset\mathrm{cl}( A )\cup\mathrm{cl}( B ) }x∈cl(B)⊂cl(A)∪cl(B)を得る。

逆にx∈cl(A){ x\in\mathrm{cl}( A ) }x∈cl(A)なら、あるフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してA∈F{ A\in\mathscr{F} }A∈Fを満たす。特にA⊂A∪B{ A\subset A\cup B }A⊂A∪BよりA∪B∈F{ A\cup B\in\mathscr{F} }A∪B∈Fであるからx∈cl(A∪B){ x\in\mathrm{cl}( A\cup B ) }x∈cl(A∪B)となる。B{ B }Bについても同様である。□{ \square }□

さて閉包がcl(cl(A))=cl(A){ \mathrm{cl}( \mathrm{cl}( A ) )=\mathrm{cl}( A ) }cl(cl(A))=cl(A)を満たせば、これは位相の公理系（Kuratowskiの公理系）を定めることになる。しかし、もちろんこれは一般に成り立たない。この公理系では閉集合をcl(A)=A{ \mathrm{cl}( A )=A }cl(A)=Aを満たす集合A⊂X{ A\subset X }A⊂Xで定めていた。

定義 X{ X }Xを収束空間とする。A⊂X{ A\subset X }A⊂Xとする。cl(A)=A{ \mathrm{cl}( A )=A }cl(A)=Aが成り立つとき、A{ A }Aは閉（closed）であるという。

補題 X{ X }Xを収束空間とする。U⊂X{ U\subset X }U⊂Xとする。以下は同値である。

  • U{ U }Uはopenである。
  • X∖U{ X\setminus U }X∖Uはclosedである。

（証明）U{ U }Uはopenとする。cl(X∖U)⊂X∖U{ \mathrm{cl}( X\setminus U )\subset X\setminus U }cl(X∖U)⊂X∖Uを示せば良い。x∈cl(X∖U){ x\in \mathrm{cl}( X\setminus U ) }x∈cl(X∖U)に対し、あるフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してX∖U∈F{ X\setminus U\in\mathscr{F} }X∖U∈Fとなる。x∈U{ x\in U }x∈UならU∈Nx⊂F{ U\in\mathscr{N}_{x}\subset\mathscr{F} }U∈Nx​⊂Fより矛盾する。よってX∖U{ X\setminus U }X∖Uはclosedである。

逆にU{ U }Uはopenでないとする。ある元x∈U{ x\in U }x∈UとフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してU∉F{ U\notin\mathscr{F} }U∈/​Fが成り立つ。F∖U={V:∃F∈F,F∖U⊂V}{ \mathscr{F}_{\setminus U}=\lbrace V : \exists F\in\mathscr{F}, F\setminus U\subset V \rbrace }F∖U​={V:∃F∈F,F∖U⊂V}はF{ \mathscr{F} }Fを含むフィルターである。X∖U∈F∖U{ X\setminus U\in\mathscr{F}_{\setminus U} }X∖U∈F∖U​かつF∖U→x{ \mathscr{F}_{\setminus U}\rightarrow x }F∖U​→xよりx∈cl(X∖U){ x\in\mathrm{cl}( X\setminus U ) }x∈cl(X∖U)を得る。x∉X∖U{ x\notin X\setminus U }x∈/​X∖UよりX∖U{ X\setminus U }X∖Uはclosedでない。□{ \square }□

収束空間がトポロジカルなときopenと開集合であることは同値だった。従って補題よりA{ A }Aが閉集合であることと、A{ A }Aがclosedであることは同値となる。よって次が成り立つ。

命題 X{ X }Xをトポロジカルな収束空間とする。閉包は位相空間の閉包と一致する。

（証明）A⊂X{ A\subset X }A⊂Xに対し、位相空間の閉包をA‾{ \overline{A} }Aで表す。

A⊂A‾{ A\subset\overline{A} }A⊂Aより、A‾{ \overline{A} }Aはclosedだからcl(A)⊂cl(A‾)=A‾{ \mathrm{cl}( A )\subset\mathrm{cl}( \overline{A} )=\overline{A} }cl(A)⊂cl(A)=Aが成り立つ。

逆を示すためにx∈X∖cl(A){ x\in X\setminus\mathrm{cl}( A ) }x∈X∖cl(A)とする。x∈A‾{ x\in\overline{A} }x∈Aとすると、任意のN∈N(x){ N\in\mathfrak{N}( x ) }N∈N(x)についてN∩A≠∅{ N\cap A\neq\emptyset }N∩A̸​=∅が成り立つ。このときフィルター⟨A⟩{ \langle A \rangle }⟨A⟩についてvel積G:=N(x)∨⟨A⟩{ \mathscr{G}:=\mathfrak{N}( x )\vee\langle A \rangle }G:=N(x)∨⟨A⟩が定義できる。今x∉cl(A){ x\notin\mathrm{cl}( A ) }x∈/​cl(A)より任意のフィルターF→x{ \mathscr{F}\rightarrow x }F→xについてA∉F{ A\notin\mathscr{F} }A∈/​Fが成り立つが、G⊃N(x)→x{ \mathscr{G}\supset\mathfrak{N}( x )\rightarrow x }G⊃N(x)→xより矛盾する。従ってx∉A‾{ x\notin\overline{A} }x∈/​Aである。□{ \square }□

以上により、トポロジカルな収束空間において、近傍、開、閉、閉包の概念は全て同一のものであることが分かる。

命題 X{ X }Xを収束空間とする。x∈X,N⊂X{ x\in X, N\subset X }x∈X,N⊂Xとする。以下は同値である。

  • N{ N }Nはx{ x }xの近傍である。
  • x∉cl(X∖N){ x\notin\mathrm{cl}( X\setminus N ) }x∈/​cl(X∖N)である。

（証明）x∈cl(X∖N){ x\in\mathrm{cl}( X\setminus N ) }x∈cl(X∖N)とする。あるフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してX∖N∈F{ X\setminus N\in\mathscr{F} }X∖N∈Fを満たす。N∉F{ N\notin\mathscr{F} }N∈/​FよりN∉Nx{ N\notin\mathscr{N}_{x} }N∈/​Nx​である。

逆にN∉Nx{ N\notin\mathscr{N}_{x} }N∈/​Nx​なら、あるフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してN∉F{ N\notin\mathscr{F} }N∈/​Fとなる。ここでF∖N={V:∃F∈F,F∖N⊂V}{ \mathscr{F}_{\setminus N}=\lbrace V : \exists F\in\mathscr{F}, F\setminus N\subset V \rbrace }F∖N​={V:∃F∈F,F∖N⊂V}はF{ \mathscr{F} }Fを含むフィルターとなる。X∖N∈F∖N{ X\setminus N\in\mathscr{F}_{\setminus N} }X∖N∈F∖N​かつF∖N→x{ \mathscr{F}_{\setminus N}\rightarrow x }F∖N​→xよりx∈cl(X∖N){ x\in\mathrm{cl}( X\setminus N ) }x∈cl(X∖N)が従う。□{ \square }□

定理 f​:X→Y{ f\colon X\rightarrow Y }f:X→Yは収束空間の間の連続写像とする。このとき以下が成り立つ。

  1. U⊂Y{ U\subset Y }U⊂Yがopenならf−1(U){ f^{-1}( U ) }f−1(U)はopenである。
  2. F⊂Y{ F\subset Y }F⊂Yがclosedならf−1(F){ f^{-1}( F ) }f−1(F)はclosedである。
  3. x∈X{ x\in X }x∈Xについて、N⊂Y{ N\subset Y }N⊂Yがf(x){ f( x ) }f(x)の近傍ならf−1(N){ f^{-1}( N ) }f−1(N)はx{ x }xの近傍である。
  4. A⊂X{ A\subset X }A⊂Xについてf(cl(A))⊂cl(f(A)){ f( \mathrm{cl}( A ) )\subset\mathrm{cl}( f( A ) ) }f(cl(A))⊂cl(f(A))が成り立つ。

（証明）1は前節で示した。2は1とf−1(Y∖F)=X∖f−1(F){ f^{-1}( Y\setminus F )=X\setminus f^{-1}( F ) }f−1(Y∖F)=X∖f−1(F)より分かる。

3を示そう。F→x{ \mathscr{F}\rightarrow x }F→xならf∗F→f(x){ f_{\ast}\mathscr{F}\rightarrow f( x ) }f∗​F→f(x)だから、Nf(x)⊂f∗F{ \mathscr{N}_{f( x )}\subset f_{\ast}\mathscr{F} }Nf(x)​⊂f∗​Fである。よってN∈Nf(x){ N\in\mathscr{N}_{f( x )} }N∈Nf(x)​について、あるF∈F{ F\in\mathscr{F} }F∈Fが存在してf(F)⊂N{ f( F )\subset N }f(F)⊂Nとなる。F⊂f−1(N){ F\subset f^{-1}( N ) }F⊂f−1(N)よりf−1(N)∈F{ f^{-1}( N )\in\mathscr{F} }f−1(N)∈Fを得る。従ってf−1(N)∈Nx{ f^{-1}( N )\in\mathscr{N}_{x} }f−1(N)∈Nx​が成り立つ。

4を示そう。x∈cl(A){ x\in\mathrm{cl}( A ) }x∈cl(A)とすると、あるフィルターF→x{ \mathscr{F}\rightarrow x }F→xが存在してA∈F{ A\in\mathscr{F} }A∈Fを満たす。f∗F→f(x){ f_{\ast}\mathscr{F}\rightarrow f( x ) }f∗​F→f(x)よりf(A)∈f(F)⊂f∗F{ f( A )\in f( \mathscr{F} )\subset f_{\ast}\mathscr{F} }f(A)∈f(F)⊂f∗​Fからf(x)∈cl(f(A)){ f( x )\in\mathrm{cl}( f( A ) ) }f(x)∈cl(f(A))を得る。□{ \square }□
